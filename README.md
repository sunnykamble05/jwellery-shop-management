#  Golden Jewellery Shop Management System

A Java-based desktop application designed to manage daily operations of a jewellery shop efficiently. This system helps in handling products, customers, billing, and inventory in a simple and organized way.

---

##  Features

-  Add, update, delete jewellery items
-  Manage inventory (gold items, stock quantity)
-  Customer management
-  Billing system with total price calculation
-  View sales records and history
-  Basic admin controls

---

##  Tech Stack

- **Language:** Java (Core Java / Swing)
- **IDE:** Eclipse / IntelliJ IDEA
- **Database:** MySQL
- **JDBC:** For database connectivity
